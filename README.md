# Mind Oasis Heart Mirror
## Project Structure
```bash
.
├── backend            # Powerd by Python
└── frontend           # Powerd by Vue.js
```
## Installation
1. Clone the repository:
```bash
git clone https://github.com/SCP173JThePeanut/A-Psychological-Assessment-System-for-Students.git
```

2. Install dependencies:
```bash
# For Python
cd backend
pip install -r requirements.txt

cd ..

#For Vue
cd frontend
npm install
```
## Usage
### Local Development
#### Using Backend
```bash
cd backend
python app.py
```
#### Using Frontend
```bash
cd frontend
npm run dev
```
